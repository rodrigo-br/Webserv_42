Seja bem-vindo ao mundo das CGIs!

CGI, ou Common Gateway Interface, é uma especificação de interface que permite que um servidor execute programas externos, como scripts, e processe requisições. Esses programas podem receber informações através do standard input ou de variáveis de ambiente.

Nosso servidor web pode executar scripts em Python proporcionando flexibilidade na criação de páginas dinâmicas.
